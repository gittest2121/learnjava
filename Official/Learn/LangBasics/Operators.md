# Operators
```
Assignment Operators -- [ = ]
Arithmetic Operators -- [ +, -, *, /, % ]
[ Arithmetic Operators + Arithmetic Operators = Compound Assignments ]
Unary Operators -- [ + (Unary plus operator -- optinally indicates positive value), - (Unary minus operator), ++ (Increment operator), -- (Decrement operator), ! (Logical complement operator -- inverts the value of a boolean) ]
Equality Operators -- [ ==, !=, >, >=, <, <= ]
Conditional Operators -- [ && (Conditional AND), || (Conditional OR) ]
Type Comparison Operator -- [ instanceof ]
Bitwise and Bit shift Operators -- [ &, ^, |, >>, <<, >>> ]
```

## Operator Precedence
```
postfix 		-- [ exp++, exp-- ]
unary			-- [ ++exp, --exp, +exp, -exp, ~, ! ]
multiplicative		-- [ *, /, % ]
additive		-- [ +, - ]
shift			-- [ <<, >>, >>> ]
relational		-- [ <, >, <=, >=, instanceof ]
equality		-- [ ==,!= ]
bitwise AND		-- [ & ]
bitwise exclusive OR	-- [ ^ ]
bitwise inclusive OR	-- [ | ]
logical AND		-- [ && ]
logical OR		-- [ || ]
ternary			-- [ ?, : ]
assignment		-- [ =, +=. -=. *=, /=, %=, ^=, !=, <<=, >>=, >>>= ]
```
